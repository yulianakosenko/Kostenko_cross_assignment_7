# AI Meal Planner

A premium React Native meal planning application with API integration, AI-inspired meal generation flow, shopping planning, and nutrition tracking.

## Features

- AI-inspired meal generation flow
- Dynamic meal planner UI
- API integration with TheMealDB
- Meal details screen
- Horizontal FlatList rendering
- Loading state handling
- Error handling
- React Navigation integration
- Reusable component architecture
- Persistent meal history
- Shopping list screen
- Responsive premium mobile UI

## Technologies

- React Native
- Expo
- React Navigation
- AsyncStorage
- FlatList
- JavaScript
- TheMealDB API

## API

The project uses:

https://www.themealdb.com/api.php
